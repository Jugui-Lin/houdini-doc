= Labs Group Invert =

"""Invert any number of specified groups"""

This node simply inverts any specified vertex, point or primitive groups.


@parameters

    Vertex Groups:
        #id: vertexgroups
        The vertex groups to be inverted.
    Point Groups:
        #id: pointgroups
        The point groups to be inverted.
    Primitive Groups:
        #id: primgroups
        The primitive groups to be inverted.
